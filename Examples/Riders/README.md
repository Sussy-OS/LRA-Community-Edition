# How to create a custom rider
### Step 1
Navigate to your documents folder > LRA and add a new folder called *Riders*.

### Step 2
In the *Riders* folder, add a new folder with the name set as what you wish to call your rider.

### Step 3
In this folder, paste and edit the files from */src/Resources/rider* in this repository to create a new rider.

### Step 4
You can now select the rider in LRA:CE in Preferences > LRTran > Rider Settings > Selected Rider.
